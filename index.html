FAQ / Help

How do I convert a file?

Select the category from the sidebar, drag and drop your file, choose the format, and click Convert.

Is it secure?

Yes! Everything runs locally on your machine. No files are uploaded to the internet.

What if conversion fails?

Check if the file is corrupted. For media files, ensure FFmpeg is installed correctly.

Can I download YouTube videos?

Yes, use the URL Downloader tab. Paste the link and choose Audio or Video mode.

What is Permanent Mode?

It creates a script to run the converter silently in the background when you start your computer.

How to create an EXE?

Go to Settings and click 'Create'. It compiles the app into a standalone executable file.

Supported formats?

We support MP3, MP4, PNG, JPG, PDF, DOCX, and many more advanced formats like FLAC, MKV, WEBP.

Is it free?

Yes, this is a free and open-source tool running on your local machine.

System Requirements?

Windows 10/11, Python 3.x, FFmpeg.
